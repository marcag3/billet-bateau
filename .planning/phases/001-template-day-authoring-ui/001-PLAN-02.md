<objective>
Provide the program-scoped entry points to create and access template days (routes + navigation + list + create).
</objective>

<must_haves>

- Template day list with "Add template day" and empty state.
</must_haves>

- Assets: staff-only template definitions and internal notes.
- Threats:
  - Accidental edits in wrong program scope (mitigate: rely on existing `requiresSelectedProgram` + program scoped collections).
  - Data loss due to navigation away mid-edit (mitigate: optimistic writes + explicit save semantics if needed; show outbox errors).
</threat_model>
