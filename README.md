Local-first pwa based on tanstack-db + electric-sql sync. Using quasar.dev components and laravel backend.

# Implementation steps:

## Priority:

[] Design architecture for separating responsability cleanly in the frontend
[] User can create template day with defined trips
[] User can assign template days to calendar day
[] User can manually add trips to calendar day

## Later:

[] User can create weekly schedule. Assign a start and end date. Assign template days or individual trips.
[] User can override weekly schedule with template days (ex: Hollidays)
