# commit

Commit your changes using https://gitmoji.dev/api/gitmojis to choose the proper emoji.

Example of usage

<intention> [scope?][:?] <message>

    intention: An emoji from the list. unicode, not shortcode
    scope: An optional string that adds contextual information for the scope of the change.
    message: A brief explanation of the change. Our conversation name can be used directly most of the time. Don't explain why.
